## 👩‍💼 Обо мне
* 🔭  Меня зовут Ирина, я занимаюсь веб-разработкой.
* 💡  При создании сайтов мне больше всего нравится видеть, как пустая страница превращается в полноценный продукт, и быть участником этой трансформации.
* 📚  Мой процесс обучения включал: тренажеры [HTML Academy](https://htmlacademy.ru/), курс [Udemy](https://www.udemy.com) по JavaScript,  [курс Result University](https://www.youtube.com/watch?v=1rRD9uMF92o&t=49693s) по **Vue 3**, материалы из открытого доступа по **React**.
* 🤖  Использую AI-инструменты для ускорения разработки.
* 🧠  В данный момент изучаю **TypeScript** в **React**, работая над приложением [Todos React TS](https://todos-react-ts-navy.vercel.app/).
* 🤝  Я открыта к любому сотрудничеству, которое поможет мне усовершенствовать проекты или приобрести новый опыт в разработке. Связаться со мной можно по почте [ivoronkova538@yandex.ru](mailto:ivoronkova538@yandex.ru).
